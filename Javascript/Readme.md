# Manipulation de Tableaux en JavaScript - Snippets
## Liste des Préfixes et Descriptions

| Préfixe               | Description |
|----------------------|-------------|
| `.array append`      | Ajoute un élément à la fin du tableau |
| `.array insert`      | Insère un élément à une position spécifique du tableau |
| `.array remove`      | Supprime la première occurrence d'un élément du tableau |
| `.array pop`         | Supprime et renvoie un élément à une position spécifique |
| `.array extend`      | Ajoute tous les éléments d'un autre tableau à la fin du tableau |
| `.array index`       | Trouve l'index de la première occurrence d'un élément dans le tableau |
| `.array count`       | Compte le nombre d'occurrences d'un élément dans le tableau |
| `.array sort`        | Trie les éléments du tableau en ordre croissant |
| `.array reverse`     | Inverse l'ordre des éléments dans le tableau |
| `.array len`         | Renvoie la longueur du tableau |
| `.array clear`       | Supprime tous les éléments du tableau |
| `.array copy`        | Crée une copie du tableau |
| `.array min`         | Trouve la valeur minimale dans le tableau |
| `.array max`         | Trouve la valeur maximale dans le tableau |
| `.array sum`         | Calcule la somme des éléments du tableau |
| `.array slice`       | Extrait une portion du tableau (slicing) |
| `.array all`         | Vérifie si tous les éléments satisfont une condition |
| `.array any`         | Vérifie si au moins un élément satisfait une condition |
| `.array zip`         | Combine deux tableaux élément par élément |
| `.array map`         | Applique une fonction à chaque élément du tableau |
| `.array filter`      | Filtre les éléments du tableau selon une condition |
